# Desafio Módulo 04 - GoNative

### Funções

- Integração com api externa.
- Filtrando produtos por categorias.
- Gravado itens no carrinho de compras.
- Itens no carrinho ficam armazenados (Redux Persist).
- Rotas e navegação de telas.
- Redux e Redux Saga.
- Padrão Duck Pattern.
